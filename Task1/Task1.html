Office To Do List Application

Frontend Development | Task Management System

I developed an interactive To-Do List Application specifically designed for office team task management. This project allows users to seamlessly add, track, edit, and delete tasks in a visually organized interface.

The application automatically segregates tasks into four distinct categories: New Task, Scheduled Task, In-Progress Task, and Completed Task, enabling smooth task tracking and prioritization. The application is built using HTML, CSS, and JavaScript, with a focus on clean UI design and practical usability.

Each task card displays the task details, assigned team member, and due date, making it easy to monitor team progress at a glance. It features an add new task functionality, a search option for quick filtering, and fully working edit and delete options to manage task updates and removals.

The project enhances user experience by providing real-time updates and easy task transitions between categories.

Through this project, I gained hands-on experience in DOM manipulation, local storage management, and dynamic content rendering. It also helped me strengthen my skills in problem-solving, critical thinking, and user-centric design.
